# 今天吃啥-校园版

## 项目简介
一个校园美食推荐系统，帮助同学们解决"今天吃什么"的难题

## 功能特性
- 🍽️ 校园周边餐厅数据导入与展示
- 🔍 多种条件筛选（价格、口味、评分等）
- 💰 基于评分和价格的"性价比"计算
- 📱 一日三餐智能推荐
- ⭐ 收藏喜欢的餐厅

## 技术栈
- Java 基础语法
- 面向对象编程
- ArrayList 集合
- MySQL 数据库

## 学习目标
- [ ] 掌握 Java 基础语法（变量、分支、循环、数组、方法）
- [ ] 掌握面向对象基础（class、对象、构造方法、封装）
- [ ] 会使用 ArrayList 存储和操作对象列表
- [ ] 独立完成《今天吃啥-校园版》控制台项目
- [ ] 初步了解 SQL，能用 MySQL 手动增删改查
- [ ] 写一份项目说明，用于转专业面试
